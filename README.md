## ✨ Overview

Nimbus is a sleek, modern, and responsive weather application designed to deliver **accurate real-time weather data** with a visually appealing interface.

It combines **performance, usability, and clean design** to provide a seamless weather tracking experience.

---

## 🚀 Features

✔️ Real-time weather updates
✔️ Global city search 🌍
✔️ 5-day forecast 📅
✔️ Air Quality Index (AQI) 🌫️
✔️ Sunrise & Sunset timing 🌅
✔️ Wind speed & direction 💨
✔️ Humidity, pressure & visibility 💧
✔️ Cloud coverage ☁️
✔️ Location details (lat, long, timezone) 📍
✔️ Clean glassmorphism UI ✨

## ⚙️ How It Works

```text
User Input → API Call → Data Processing → UI Update
```

1. User enters a city
2. App fetches weather data via API
3. Data is processed and displayed
4. UI updates dynamically
